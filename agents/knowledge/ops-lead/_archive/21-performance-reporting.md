## 1. 성과 리포트의 목적

- 약속한 것을 했는지 증명
- 무엇이 효과 있고 없는지 파악
- 다음 기간 전략 수립 근거
- 클라이언트와의 신뢰 구축

---

## 2. 리포트 주기별 구성

### 주간 리포트 (가벼움, 1~2페이지)
```markdown
## 주간 업데이트 (3/11~3/15)

### 이번 주 주요 성과
- 블로그 발행: 2편 (계획 2편 ✅)
- 주요 키워드 "마케팅 자동화" 7위 → 5위 상승
- 인스타 팔로워 +85 (이번 주 최고 성장)

### 주목할 콘텐츠
- "콘텐츠 마케팅 ROI 측정법" — 발행 3일 만에 조회수 1,200

### 다음 주 계획
- 블로그 2편 발행 예정 (주제 첨부)
- 이메일 뉴스레터 발송 (목요일)
```

### 월간 리포트 (상세, 5~10페이지)
```markdown
## 3월 월간 성과 리포트

1. 핵심 요약 (Executive Summary)
2. 목표 달성 현황 (KPI 대시보드)
3. 채널별 상세 성과
   - SEO / 블로그
   - SNS
   - 이메일
4. 주요 콘텐츠 성과 TOP 5
5. 경쟁사 동향 (선택)
6. 인사이트 및 다음 달 전략
7. 부록 (원시 데이터)
```

## 3. 데이터 스토리텔링

```
나쁜 리포트:
  "3월 오가닉 트래픽은 14,600이었습니다."

좋은 리포트:
  "3월 오가닉 트래픽은 14,600으로, 목표(13,000) 대비 112% 달성했습니다.
  특히 2월 대비 +30% 성장은 1~2월 집중 발행한
  롱테일 키워드 콘텐츠 12편이 검색 순위에 반영되기 시작한 결과입니다.
  이 추세가 지속되면 4월에는 자연스럽게 15,000 이상 달성이 예상됩니다."
```

**스토리텔링 구조:**
1. 수치 (What)
2. 맥락 (Context) — 목표 대비, 전월 대비
3. 원인 (Why) — 무엇이 이 결과를 만들었는가
4. 의미 (So What) — 비즈니스에 어떤 의미인가
5. 다음 (Now What) — 무엇을 할 것인가

## 4. 리포트 자동화

```
Google Looker Studio:
  → 클라이언트가 직접 실시간 확인 가능
  → 월간 리포트는 PDF 스냅샷으로 발송

Google Sheets + Apps Script:
  → 자동 데이터 수집
  → 자동 차트 업데이트
  → 자동 이메일 발송

리포트 자동화 ROI:
  수동 리포트 작성: 클라이언트당 3~4시간/월
  자동화 후: 30분~1시간/월 (검토 + 인사이트 작성)
```

## 5. 목표 미달 시 리포트

```
솔직하되 건설적으로:

✅ 좋은 예:
"3월 오가닉 트래픽은 11,200으로, 목표(13,000) 대비 86% 달성했습니다.
3월 초 구글 코어 업데이트로 업계 전반적으로 10~15% 감소가 있었으며,
저희도 영향을 받았습니다. (업계 평균: -12%, 저희: -5%)

4월은 업데이트 적응 콘텐츠 3편 + E-E-A-T 강화 작업을 통해
회복 및 목표 달성을 목표로 합니다."

❌ 나쁜 예:
"구글 업데이트 때문에 목표를 달성하지 못했습니다.
다음 달엔 더 잘 하겠습니다."
```

## 6. 안티패턴

- **수치 나열만**: 해석 없는 숫자 → 클라이언트가 판단 불가
- **좋은 지표만 강조**: 신뢰 손상 → 미달 지표도 원인과 함께
- **리포트 지연**: 다음 달에 지난 달 리포트 → 가치 감소
- **클라이언트 맞춤화 없음**: 모든 클라이언트에게 동일한 리포트 형식
- **다음 액션 없는 리포트**: 과거만 분석, 미래 전략 없음
