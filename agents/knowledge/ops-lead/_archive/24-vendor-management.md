## 1. 벤더 유형

```
프리랜서/외주 개인:
  작가, 디자이너, 번역가, 영상 편집자

에이전시/회사:
  SEO 전문 에이전시, 미디어 구매, PR

SaaS 도구:
  마케팅 자동화, 분석 도구, 협업 도구

플랫폼:
  광고 플랫폼, 콘텐츠 배포 플랫폼
```

---

## 2. 벤더 선정 기준

```
평가 항목:

품질:
  - 포트폴리오 검토
  - 레퍼런스 체크 (실제 사용자 피드백)
  - 테스트 작업 진행

신뢰성:
  - 납기 준수 이력
  - 소통 응답 속도
  - 문제 발생 시 대응 방식

비용:
  - 단순 최저가 선택 금지
  - 품질/시간 절약 대비 ROI 계산

리스크:
  - 단일 벤더 의존도
  - 계약서/NDA 가능 여부
  - 정보 보안
```

---

## 3. 프리랜서 관리

```
온보딩:
  - 브랜드 가이드라인 공유
  - 작업 방식 가이드 문서
  - 첫 작업은 소규모로 시작

계약:
  - 업무 범위 명확히
  - 납기 및 수정 횟수 명시
  - 저작권 귀속 (클라이언트 소유)
  - NDA (민감한 클라이언트 정보)

퍼포먼스 관리:
  - 작업별 피드백 제공
  - 품질 등급 기록 유지
  - 우수 벤더 우선 배정

지급:
  - 명확한 인보이스 기준
  - 지급 일정 사전 합의
  - 세금계산서 요청
```

## 4. 벤더 레지스터

```markdown
## 벤더 레지스터

| 이름 | 유형 | 전문 분야 | 단가 | 품질 등급 | 마지막 작업 | 비고 |
|------|------|---------|------|---------|-----------|------|
| 홍작가 | 프리랜서 | SEO 블로그 | 5만/1000자 | A | 2024-03 | 마감 잘 지킴 |
| 김디자인 | 프리랜서 | SNS 카드뉴스 | 8만/세트 | B | 2024-02 | 수정 잦음 |
| OO에이전시 | 에이전시 | 백링크 구축 | 월 150만 | A | 진행중 | 재계약 예정 |

품질 등급:
  A: 재작업 없이 납품, 응답 빠름, 마감 준수
  B: 가끔 수정 필요, 보통 수준
  C: 지속 피드백 필요, 마지막 수단
```

---

## 5. 벤더 리스크 관리

```
단일 벤더 의존 방지:
  - 핵심 역할은 최소 2~3개 벤더 유지
  - 주요 벤더 작업 불가 시 대안 항상 준비

정보 보안:
  - 클라이언트 기밀 정보는 NDA 없이 공유 금지
  - 로그인 정보는 1Password 팀 계정으로 공유
  - 프로젝트 종료 후 접근 권한 회수

품질 저하 대응:
  - 2회 연속 C등급 → 새 벤더 탐색 시작
  - 핵심 작업은 B 이상만 배정
```

## 6. 안티패턴

- **가격만 보는 선택**: 최저가 = 최저 품질, 재작업 비용 포함 계산
- **계약서 없는 외주**: 분쟁 시 근거 없음
- **온보딩 없는 작업 배정**: 브랜드 이해 없이 → 부적절한 결과물
- **피드백 없는 관계**: 개선 기회 없음 → 계속 같은 수준
- **벤더 레지스터 없음**: 매번 새로 찾음 → 비효율
