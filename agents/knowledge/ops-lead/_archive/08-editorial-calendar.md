## 1. 에디토리얼 캘린더란

언제, 어떤 채널에, 어떤 콘텐츠를 발행할지 계획하는 문서.
임기응변 콘텐츠 → 전략적 콘텐츠 발행으로.

---

## 2. 캘린더 구성 요소

```markdown
## 에디토리얼 캘린더 — 2024년 4월

| 날짜 | 채널 | 제목/주제 | 목표 키워드 | 담당자 | 상태 |
|------|------|---------|-----------|------|------|
| 4/1 | 블로그 | "스타트업 SEO 전략 A to Z" | 스타트업 SEO | 홍길동 | 작성 중 |
| 4/3 | 인스타 | 봄 시즌 신제품 소개 | - | 김철수 | 기획 |

| 4/8 | 블로그 | "콘텐츠 마케팅 ROI 측정법" | 콘텐츠마케팅 ROI | 홍길동 | 예정 |
| 4/10 | 유튜브 | 제품 사용법 튜토리얼 | - | 김철수 | 기획 |
```

## 3. 캘린더 기획 프로세스

```
1. 월간 목표 확인 (KPI 기반)
   → 이번 달 무엇을 달성해야 하는가?

2. 이벤트/시즌 파악
   → 공휴일, 업계 이벤트, 제품 출시 일정

3. 콘텐츠 믹스 결정
   → SEO 포스트 : 소셜 : 이메일 = ?

4. 주제/키워드 선정
   → 키워드 리서치, 경쟁사 분석

5. 발행 일정 배분
   → 채널별 최적 발행 빈도 고려

6. 담당자 배정 및 기한 설정
   → 역방산으로 작업 기한 산출
```

## 4. 콘텐츠 믹스 전략

```
80/20 법칙:
  80% — 가치 제공 콘텐츠 (정보, 교육, 엔터테인먼트)
  20% — 홍보 콘텐츠 (제품, 서비스 소개)

콘텐츠 유형 믹스:
  - 에버그린 (Evergreen): 시간이 지나도 유효 (SEO 효과 지속)
  - 트렌딩: 현재 이슈 반응 (단기 트래픽)
  - 시즌성: 특정 시기에만 관련 (기획 미리 필요)
  - 프로모션: 제품/서비스 홍보
```

## 5. 최적 발행 시간 (채널별)

```
인스타그램:
  - 평일 오전 9~11시
  - 점심 12~1시
  - 저녁 7~9시

블로그/SEO:
  - 요일 영향 낮음 (검색 유입)
  - 평일 아침 발행 권장 (인덱싱 + 첫날 트래픽)

이메일:
  - 화/수/목 오전 10~11시
  - 월요일, 금요일 피하기

링크드인:
  - 화~목 오전 8~10시
  - 점심 12시
```

## 6. 캘린더 관리 도구

```
Google Sheets — 단순하고 공유 쉬움
Notion — 캔반 뷰 + 데이터베이스
Airtable — 강력한 필터/그룹핑
CoSchedule — 전용 에디토리얼 도구

최소 구성:
  구글 시트 + 구글 캘린더
  → 시트: 상세 계획
  → 캘린더: 발행 일정 시각화
```

---

## 7. 안티패턴

- **캘린더 없는 즉흥 발행**: 일관성 없는 발행 → 알고리즘 불이익
- **너무 촘촘한 계획**: 갑작스러운 변경에 대응 어려움 → 20% 여유
- **캘린더 공유 안 함**: 팀원/클라이언트가 계획 모름
- **실적 업데이트 없음**: 계획만 있고 실제 발행 현황 추적 안 함
- **한 채널만 계획**: 크로스 채널 시너지 미고려
