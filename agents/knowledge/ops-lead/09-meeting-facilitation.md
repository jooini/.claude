## 1. 효과적인 미팅의 조건

- **목적이 명확**: 결정, 정보 공유, 아이디어 발산 중 무엇인가?
- **필요한 사람만**: 의사결정 권한 있는 사람, 직접 관련된 사람만
- **준비된 참석자**: 아젠다 사전 공유, 자료 미리 배포
- **시간 엄수**: 시작/종료 시간 준수
- **액션으로 끝남**: 담당자+기한이 명시된 액션 아이템

---

## 2. 미팅 전 준비

```markdown
## 미팅 준비 체크리스트

[ ] 목적 명확화: 이 미팅으로 무엇을 결정/달성해야 하는가?
[ ] 참석자 선정: 꼭 필요한 사람만 (5명 초과 시 재검토)
[ ] 아젠다 작성: 항목별 시간 배분 포함
[ ] 사전 자료 공유: 최소 24시간 전
[ ] 시간 확인: 모든 참석자 가능한 시간
[ ] 장소/링크: Zoom/Meet 링크 공유
```

**미팅이 필요한가 판단:**
- 이메일/메신저로 해결 가능? → 미팅 불필요
- 1:1로 해결 가능? → 대규모 미팅 불필요
- 정보 전달만? → 녹화 영상 또는 문서로

## 3. 아젠다 작성

```markdown
# 정기 주간 미팅 아젠다
날짜: 3/15 (금) 오후 2시 (30분)

참석: 홍길동(PM), 김철수(콘텐츠), 이영희(디자인)

## 1. 이번 주 진행 상황 (10분)
   - 각자 완료/진행 중/블로킹 사항 공유

## 2. 결정 필요 사항 (15분)
   2-1. A사 4월 콘텐츠 방향 결정 (클라이언트 피드백 반영)
   2-2. B사 이미지 예산 추가 승인 여부

## 3. 다음 주 계획 확인 (5분)

## 사전 준비
- 각자 이번 주 작업 현황 정리
- A사 피드백 원문: [링크]
```

## 4. 퍼실리테이션 기술

**시작 설정:**
```
"오늘 미팅은 [목적]을 위해 [시간] 동안 진행합니다.
오늘 결정해야 할 것은 [결정 사항]입니다.
[이름]님이 기록을 맡아주시면 감사하겠습니다."
```

**논의 정리:**
```
"지금까지 나온 의견을 정리하면... 맞나요?"
"두 가지 방향이 나왔는데, [A안] vs [B안]으로 정리됩니다."
```

**시간 관리:**
```
"이 부분은 중요해서 더 논의가 필요할 것 같습니다.
오늘은 [핵심 결정]만 하고, 세부 사항은 별도로 논의할까요?"
```

**침묵하는 참석자 참여 유도:**
```
"[이름]님은 어떻게 생각하시나요?"
"[이름]님이 이 부분 가장 잘 아실 것 같은데, 의견 부탁드립니다."
```

## 5. 미팅 노트 작성

```markdown
# 미팅 노트 — 2024/3/15 주간 미팅

참석: 홍길동, 김철수, 이영희
기록: 이영희

## 논의 내용

### A사 4월 콘텐츠 방향
- 클라이언트 피드백: SEO 보다 브랜드 인지도 강화 원함
- 논의: SEO + 브랜드 균형 (7:3 → 5:5로 조정)
- **결정**: 4월부터 SEO : 브랜드 = 5:5 비율 적용

### B사 이미지 예산
- 현재 50만원 → 추가 30만원 요청
- **결정**: PM이 클라이언트에게 확인 후 결정 (기한: 3/17)

## 액션 아이템
| 항목 | 담당 | 기한 |
|------|------|------|
| A사 4월 캘린더 수정 | 김철수 | 3/18 |
| B사 예산 클라이언트 확인 | 홍길동 | 3/17 |
| 4월 디자인 리소스 확인 | 이영희 | 3/17 |

## 다음 미팅
3/22 (금) 오후 2시
```

---

## 6. 안티패턴

- **아젠다 없는 미팅**: 방향 없이 흘러감
- **결정 없이 끝남**: "다음에 다시 얘기해요" → 같은 논의 반복
- **액션 아이템 없음**: 미팅 내용이 실행으로 이어지지 않음
- **너무 많은 참석자**: 의사결정 속도 저하, 일부는 시간 낭비
- **미팅 노트 미공유**: 참석 못한 사람, 나중에 기억 못하는 상황
