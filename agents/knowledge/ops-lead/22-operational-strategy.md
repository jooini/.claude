## 1. 운영 전략이란

일상 운영(Tactics)을 넘어 **어떻게 운영할 것인가**에 대한 중장기 방향.
비즈니스 목표 → 운영 방식으로 번역하는 것.

---

## 2. 운영 전략 수립 프레임워크

```
비즈니스 목표
  "연간 매출 30억 달성"
      ↓
운영 목표
  "클라이언트 20개사 유지, 평균 계약금 1.5억"
      ↓
운영 역량
  "월 50개 이상 콘텐츠 제작, 평균 납기 준수율 95%"
      ↓
운영 구조
  팀 구성, 프로세스, 도구, 파트너십
```

---

## 3. 운영 모델 선택

```
자체 운영 모델:
  모든 것을 내부 팀으로
  → 높은 통제력, 낮은 유연성
  → 고정 인건비 높음

하이브리드 모델 (권장):
  핵심 역량: 내부 (전략, QC, 클라이언트 관리)
  실행: 내부 + 프리랜서 조합
  → 적정 통제력 + 유연성

아웃소싱 중심:
  대부분 외주
  → 낮은 고정비, 품질 통제 어려움
```

---

## 4. 운영 효율화 로드맵

```
현재 (1~3명 팀):
  - 모든 것을 손으로
  - 체크리스트 기반 품질 관리
  - 기본 협업 도구

성장기 (4~8명):
  - 역할 전문화 (콘텐츠, 디자인, 분석)
  - 워크플로우 도구 도입
  - SOP 문서화

확장기 (10명+):
  - 팀장 레이어 추가
  - 자동화 투자
  - 데이터 기반 의사결정

각 단계에서 먼저 해야 할 것:
  사람 추가 전에 프로세스 정리
  도구 추가 전에 사용법 숙달
```

---

## 5. 경쟁 우위 전략

```
차별화 포인트 선택:

비용 우위 전략:
  - 자동화 최대화로 저비용 운영
  - 대규모 클라이언트에 적합

차별화 전략:
  - 특정 업종 전문화 (예: B2B SaaS 마케팅)
  - 프리미엄 품질 + 높은 가격
  - 결과 보장형 계약

집중화 전략:
  - 특정 니치(niche) 시장만 공략
  - 깊은 전문성으로 경쟁
```

---

## 6. 안티패턴

- **전략 없는 운영**: 바쁘게 일하지만 방향성 없음
- **전략만 있고 실행 없음**: 좋은 계획도 실행 없으면 무의미
- **전략 변경 남발**: 매 분기 다른 방향 → 팀 혼선
- **운영 지표 없는 전략**: 측정 못하면 달성 여부 모름
- **경쟁 분석 없는 차별화**: "우리는 특별하다" → 근거 없음
